# CrymadX - Cryptocurrency Trading Platform

A modern, feature-rich cryptocurrency trading platform UI built with React, TypeScript, and Vite.

## Features

- **Dashboard** - Overview of portfolio, markets, and trading activity
- **Trading** - Real-time trading interface with charts
- **Wallets** - Multi-currency wallet management
- **P2P Trading** - Peer-to-peer cryptocurrency exchange
- **Savings** - Earn interest on crypto holdings
- **Referral System** - Invite friends and earn rewards

## Tech Stack

- React 18
- TypeScript
- Vite
- Framer Motion (animations)
- Glassmorphism UI design

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured for deployment on Vercel.
